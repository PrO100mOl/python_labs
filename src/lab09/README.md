# Лабораторная работа 9 — CSV-хранилище студентов

## Класс `Group`
- Инкапсулирует CRUD-операции над студентами, сохранёнными в CSV (`fio,birthdate,group,gpa`).
- Использует `Student` из ЛР8, поэтому вся валидация формата даты и диапазона GPA сохраняется.
- Базовые методы:
  - `list()` — чтение всех записей.
  - `add(student)` — добавление нового студента.
  - `find(substr)` — поиск по подстроке в `fio` (без учёта регистра).
  - `remove(fio)` — удаление записей с указанным ФИО.
  - `update(fio, **fields)` — обновление полей найденного студента.
  - `stats()` — агрегированная статистика (количество, мин/макс/средний GPA, распределение по группам, топ-5).

## Пример использования
```python
from src.lab08 import Student
from src.lab09 import Group

group = Group("data/lab09/students.csv")

# Чтение и поиск
a = group.list()
print(len(a), a[0])
print(group.find("иванов"))

# Добавление
group.add(Student("Новый Студент", "2004-12-31", "BD-02", 4.1))

# Обновление и удаление
group.update("Иванов Иван", gpa=4.5)
group.remove("Сидоров Павел")

# Статистика
print(group.stats())
```
